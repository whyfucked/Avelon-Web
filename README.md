## <a name="english"></a>English

### About

**Avelon Web** is a billing system and website that was used on the Avelon project for game server hosting. Built with modern web technologies, it provides a comprehensive solution for server management, payments, and user administration.

### Tech Stack

- **Language:** TypeScript
- **Framework:** Next.js
- **Database:** MySQL (with Prisma ORM)

### Development

This project was developed by:
- **XEHKU**
- **WhyFucked** (Nikita Fialkovsky)

During development, we utilized AI assistance from **Codex** to help with certain parts of the codebase.

### Payment Methods

- **CrystalPay** - Cryptocurrency payment gateway
- **YooMoney** - Russian payment system
- **Heleket** - Multi-currency payment processor
- **FreeKassa** - Payment aggregator

### Features

- User authentication and authorization
- Server management (Pterodactyl & VMManager integration)
- Multiple payment gateways integration
- Admin panel with comprehensive controls
- Status page monitoring
- Dedicated server management
- Storage box management
- Promotional codes and plans

## <a name="russian"></a>Русский

### О проекте

**Avelon Web** — биллинг-система и веб-сайт, который использовался на провекте Avelon для хостинга игровых серверов. Построен на современных веб-технологиях и предоставляет комплексное решение для упраления серверами, платежами и администрирования пользователей.

### Технологический стек

- **Язык:** TypeScript
- **Фреймворк:** Next.js
- **База данных:** MySQL (с Prisma ORM)

### Разработка

Этот проект был разработан:
- **XEHKU**
- **WhyFucked** (Никита Фиалковский)

При написании кода использовалась помощь ИИ **Codex** для некоторых частей кодовой базы.

### Платежные системы

- **CrystalPay** - Криптовалютный платежный шлюз
- **ЮMoney** - Российская платежная система
- **Heleket** - Мультивалютный платежный процессор
- **FreeKassa** - Платежный агрегатор

### Возможности

- Аутентификация и авторизация пользователей
- Управление серверами (интеграция с Pterodactyl и VMManager)
- Интеграция с несколькими платежными шлюзами
- Админ-панель с полным набором инструментов
- Мониторинг статуса сервисов
- Управление выделенными серверами
- Управление хранилищами
- Промокоды и тарифные планы
